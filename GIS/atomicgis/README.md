# AtomicGIS

AtomicGIS is an execution-oriented GIS sandbox for AI agents.

## Features

- Atomic GIS tools
- FastAPI backend
- Workflow DAG execution
- Tool Registry
- MCP Server
- GeoAI modules
- Vue3 + MapLibre frontend
- Docker Compose support

## Quick Start

```bash
docker compose up --build
```

Backend:
http://localhost:8000

Frontend:
http://localhost:5173

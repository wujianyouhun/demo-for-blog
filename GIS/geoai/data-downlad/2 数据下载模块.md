## 核心功能概述

GeoAI 的数据下载模块 (`geoai/download.py`) 提供了以下核心功能：

1. **NAIP 影像下载**：从微软 Planetary Computer 下载美国国家农业影像计划 (NAIP) 数据
2. **Overture Maps 数据下载**：从 Overture Maps 下载建筑物、地址等开放地理空间数据
3. **Planetary Computer STAC 数据处理**：搜索、下载和可视化 STAC (SpatioTemporal Asset Catalog) 项目
4. **数据格式转换**：在不同地理空间数据格式之间进行转换
5. **数据统计与分析**：提取和分析下载数据的统计信息

## 关键函数详解

### 1. NAIP 影像下载

#### `download_naip()`

**功能**：从 Planetary Computer 下载 NAIP 影像

**参数**：
- `bbox`：边界框坐标 (min_lon, min_lat, max_lon, max_lat)
- `output_dir`：输出目录
- `year`：指定年份（可选）
- `max_items`：最大下载数量
- `overwrite`：是否覆盖现有文件
- `preview`：是否预览下载的影像

**核心代码**：

```python
def download_naip(
    bbox: Tuple[float, float, float, float],
    output_dir: str,
    year: Optional[int] = None,
    max_items: int = 10,
    overwrite: bool = False,
    preview: bool = False,
    **kwargs: Any,
) -> List[str]:
    # 创建输出目录
    os.makedirs(output_dir, exist_ok=True)
    
    # 创建边界框几何对象
    geometry = box(*bbox)
    
    # 连接到 Planetary Computer STAC API
    catalog = Client.open("https://planetarycomputer.microsoft.com/api/stac/v1")
    
    # 构建查询参数
    search_params = {
        "collections": ["naip"],
        "intersects": geometry,
        "limit": max_items,
    }
    
    # 添加年份过滤器
    if year:
        search_params["query"] = {"naip:year": {"eq": year}}
    
    # 搜索 NAIP 影像
    search_results = catalog.search(**search_params)
    items = list(search_results.items())
    
    # 下载并保存每个项目
    downloaded_files = []
    for i, item in enumerate(items):
        # 签名资产（Planetary Computer 需要）
        signed_item = pc.sign(item)
        
        # 获取 RGB 资产 URL
        rgb_asset = signed_item.assets.get("image")
        if not rgb_asset:
            logger.warning("No RGB asset found for item %d", i + 1)
            continue
        
        # 构建输出路径
        original_filename = os.path.basename(rgb_asset.href.split("?")[0])
        output_path = os.path.join(output_dir, original_filename)
        
        # 下载文件
        if rgb_asset.href.startswith("http"):
            download_with_progress(rgb_asset.href, output_path)
        else:
            data = rxr.open_rasterio(rgb_asset.href)
            data.rio.to_raster(output_path)
        
        downloaded_files.append(output_path)
    
    return downloaded_files
```

**代码说明**：
1. 创建输出目录并构建边界框几何对象
2. 连接到 Planetary Computer STAC API 并构建查询参数
3. 搜索符合条件的 NAIP 影像项目
4. 对每个项目进行签名（Planetary Computer 要求）
5. 下载 RGB 资产并保存为 GeoTIFF 文件
6. 返回下载的文件路径列表

### 2. Overture Maps 数据下载

#### `download_overture_buildings()`

**功能**：从 Overture Maps 下载建筑物数据

**参数**：
- `bbox`：边界框坐标
- `output`：输出文件路径
- `overture_type`：Overture 数据类型（默认为 "building"）

**核心代码**：

```python
def download_overture_buildings(
    bbox: Tuple[float, float, float, float],
    output: str,
    overture_type: str = "building",
    **kwargs: Any,
) -> str:
    return get_overture_data(
        overture_type=overture_type, bbox=bbox, output=output, **kwargs
    )

def get_overture_data(
    overture_type: str,
    bbox: Tuple[float, float, float, float] = None,
    columns: List[str] = None,
    output: str = None,
    **kwargs: Any,
) -> "gpd.GeoDataFrame":
    try:
        from overturemaps import core
    except ImportError:
        raise ImportError("The overturemaps package is required to use this function")
    
    gdf = core.geodataframe(overture_type, bbox=bbox)
    if columns is not None:
        gdf = gdf[columns]
    
    gdf.crs = "EPSG:4326"
    
    if output is not None:
        out_dir = os.path.dirname(os.path.abspath(output))
        if not os.path.exists(out_dir):
            os.makedirs(out_dir, exist_ok=True)
        gdf.to_file(output, **kwargs)
    
    return gdf
```

**代码说明**：
1. 导入 overturemaps 核心模块
2. 使用核心模块的 geodataframe 函数获取数据
3. 设置坐标系为 EPSG:4326
4. 保存数据到指定输出文件
5. 返回 GeoDataFrame 对象

### 3. Planetary Computer STAC 数据处理

#### `pc_stac_search()`

**功能**：在 Planetary Computer 中搜索 STAC 项目

**参数**：
- `collection`：STAC 集合 ID
- `bbox`：边界框坐标
- `time_range`：时间范围
- `query`：额外的查询参数
- `limit`：每页返回的项目数
- `max_items`：最大返回项目数

**核心代码**：

```python
def pc_stac_search(
    collection: str,
    bbox: Optional[List[float]] = None,
    time_range: Optional[str] = None,
    query: Optional[Dict[str, Any]] = None,
    limit: int = 10,
    max_items: Optional[int] = None,
    quiet: bool = False,
    endpoint: str = "https://planetarycomputer.microsoft.com/api/stac/v1",
) -> List["pystac.Item"]:
    # 初始化 STAC 客户端
    catalog = Client.open(endpoint)
    
    # 处理时间范围
    if time_range:
        if isinstance(time_range, tuple) and len(time_range) == 2:
            start, end = time_range
            if isinstance(start, datetime.datetime):
                start = start.isoformat()
            if isinstance(end, datetime.datetime):
                end = end.isoformat()
            time_str = f"{start}/{end}"
        elif isinstance(time_range, str):
            time_str = time_range
        else:
            raise ValueError("time_range must be a 'start/end' string or tuple of (start, end)")
    else:
        time_str = None
    
    # 创建搜索对象
    search = catalog.search(
        collections=[collection], bbox=bbox, datetime=time_str, query=query, limit=limit
    )
    
    # 收集项目
    items = []
    if max_items:
        items_gen = search.get_items()
        for item in items_gen:
            items.append(item)
            if len(items) >= max_items:
                break
    else:
        items = list(search.get_items())
    
    return items
```

**代码说明**：
1. 初始化 STAC 客户端并连接到 Planetary Computer API
2. 处理时间范围参数，支持字符串或 datetime 元组格式
3. 创建搜索对象并执行搜索
4. 收集搜索结果，支持限制最大返回数量
5. 返回 STAC 项目列表

#### `pc_stac_download()`

**功能**：下载 STAC 项目的资产

**参数**：
- `items`：STAC 项目或项目列表
- `output_dir`：输出目录
- `assets`：要下载的资产列表
- `max_workers`：最大并发下载线程数
- `skip_existing`：是否跳过已存在的文件

**核心代码**：

```python
def pc_stac_download(
    items: Union["pystac.Item", List["pystac.Item"]],
    output_dir: str = ".",
    assets: Optional[List[str]] = None,
    max_workers: int = 1,
    skip_existing: bool = True,
) -> Dict[str, Dict[str, str]]:
    # 处理单个项目的情况
    if isinstance(items, pystac.Item) or isinstance(items, str):
        items = [items]
    
    # 创建输出目录
    os.makedirs(output_dir, exist_ok=True)
    
    # 下载单个资产的函数
    def download_asset(item, asset_key, asset):
        item = pc.sign(item)
        item_id = item.id
        
        # 获取资产 URL 并签名
        asset_url = item.assets[asset_key].href
        
        # 确定输出文件名
        if asset.media_type:
            if "tiff" in asset.media_type or "geotiff" in asset.media_type:
                ext = ".tif"
            elif "jpeg" in asset.media_type:
                ext = ".jpg"
            elif "png" in asset.media_type:
                ext = ".png"
            elif "json" in asset.media_type:
                ext = ".json"
            else:
                ext = os.path.splitext(asset_url.split("?")[0])[1] or ".data"
        else:
            ext = os.path.splitext(asset_url.split("?")[0])[1] or ".data"
        
        output_path = os.path.join(output_dir, f"{item_id}_{asset_key}{ext}")
        
        # 跳过已存在的文件
        if skip_existing and os.path.exists(output_path):
            return asset_key, output_path
        
        # 下载资产
        with requests.get(asset_url, stream=True) as r:
            r.raise_for_status()
            total_size = int(r.headers.get("content-length", 0))
            with open(output_path, "wb") as f:
                with tqdm(
                    total=total_size,
                    unit="B",
                    unit_scale=True,
                    desc=f"Downloading {item_id}_{asset_key}",
                ) as pbar:
                    for chunk in r.iter_content(chunk_size=8192):
                        f.write(chunk)
                        pbar.update(len(chunk))
        
        return asset_key, output_path
    
    # 处理所有项目及其资产
    results = {}
    for item in items:
        item_assets = {}
        if isinstance(item, str):
            item = pystac.Item.from_file(item)
        item_id = item.id
        
        # 确定要下载的资产
        if assets:
            assets_to_download = {k: v for k, v in item.assets.items() if k in assets}
        else:
            assets_to_download = item.assets
        
        # 并发下载资产
        with ThreadPoolExecutor(max_workers=max_workers) as executor:
            future_to_asset = {
                executor.submit(download_asset, item, asset_key, asset): (asset_key, asset)
                for asset_key, asset in assets_to_download.items()
            }
            
            for future in as_completed(future_to_asset):
                asset_key, asset = future_to_asset[future]
                try:
                    key, path = future.result()
                    if path:
                        item_assets[key] = path
                except Exception as e:
                    logger.error("Error processing asset %s for item %s: %s", asset_key, item_id, str(e))
        
        results[item_id] = item_assets
    
    return results
```

**代码说明**：
1. 处理输入参数，支持单个项目或项目列表
2. 创建输出目录
3. 定义内部函数 `download_asset` 用于下载单个资产
4. 为每个项目确定要下载的资产
5. 使用 ThreadPoolExecutor 并发下载资产
6. 处理下载结果并返回资产路径映射

### 4. 辅助函数

#### `download_with_progress()`

**功能**：带进度条的文件下载

**参数**：
- `url`：文件 URL
- `output_path`：输出路径
- `max_size`：最大文件大小（可选）

**核心代码**：

```python
def download_with_progress(
    url: str, output_path: str, max_size: Optional[int] = None
) -> None:
    _validate_url(url)
    response = requests.get(url, stream=True)
    response.raise_for_status()
    total_size = int(response.headers.get("content-length", 0))
    
    if max_size is not None and total_size > max_size:
        response.close()
        raise ValueError(
            f"File size ({total_size} bytes) exceeds the maximum "
            f"allowed size ({max_size} bytes)."
        )
    
    block_size = 1024  # 1 Kibibyte
    
    with (
        open(output_path, "wb") as file,
        tqdm(
            desc=os.path.basename(output_path),
            total=total_size,
            unit="iB",
            unit_scale=True,
            unit_divisor=1024,
        ) as bar,
    ):
        for data in response.iter_content(block_size):
            size = file.write(data)
            bar.update(size)
```

**代码说明**：
1. 验证 URL 格式
2. 发送 GET 请求并获取文件大小
3. 检查文件大小是否超过限制
4. 使用 tqdm 创建进度条
5. 分块下载文件并更新进度条

## 常见问题解决

### 1. 下载速度慢

**解决方案**：
- 增加 `max_workers` 参数值（默认为 1）
- 使用较小的 `max_items` 值，分批下载
- 确保网络连接稳定

### 2. 内存不足

**解决方案**：
- 减少 `max_items` 值
- 避免一次性下载太多资产
- 使用流式处理而不是加载整个文件到内存

### 3. 认证错误

**解决方案**：
- 确保 Planetary Computer 认证正确
- 使用 `pc.sign()` 方法签名资产
- 检查网络连接和 API 访问权限

### 4. 文件格式问题

**解决方案**：
- 使用 `convert_vector_format()` 函数转换格式
- 确保输出目录存在且有写入权限
- 检查文件扩展名是否正确

### 5. 依赖包缺失

**解决方案**：
- 安装缺失的依赖包，例如：
  ```bash
  pip install overturemaps
  ```
- 对于 Overture Maps 功能，需要安装 overturemaps 包

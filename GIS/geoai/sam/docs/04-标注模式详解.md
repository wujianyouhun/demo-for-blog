## 04 - 标注模式详解

本项目支持四种标注模式，每种模式适用于不同的场景。本文档详细说明各模式的参数、技巧和最佳实践。

### 模式一: 点提示标注

点提示是最简单的交互方式：在目标上点击一个或多个点，SAM 推断出对应的分割区域。

#### 适用场景

目标边界模糊、不确定具体范围时的快速探索。例如在影像中大致定位一栋建筑、一个水体。

#### 基本用法

```python
from geoai_sam import SAMWrapper

sam = SAMWrapper(model_type="vit_h", automatic=False)
sam.set_image("西安19级.tif")

# 单点: 在目标中心点击
masks = sam.generate_masks_by_points([[750, 370]])
```

#### 多点提示

可以在多个目标上分别点击:

```python
masks = sam.generate_masks_by_points(
    points=[[500, 375], [1125, 625]],
    point_labels=[1, 1],   # 两个都是前景
)
```

#### 正负样本联合

这是点提示最强大的用法。前景点 (label=1) 告诉 SAM "这是我想要的"，背景点 (label=0) 告诉 SAM "这不是我要的":

```python
masks = sam.generate_masks_by_points(
    points=[
        [750, 370],    # 建筑物 → 前景
        [1125, 625],   # 树木 → 背景 (排除)
    ],
    point_labels=[1, 0],
)
```

当 SAM 的默认分割结果包含了不想要的区域时，在那些区域添加背景点可以有效修正。

#### 参数说明

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `points` | List[List[int]] | 必填 | 坐标列表 [[x,y], ...] |
| `point_labels` | List[int] | None (全前景) | 标签列表，1=前景 0=背景 |
| `multimask_output` | bool | True | True 输出 3 个候选 mask，False 输出 1 个 |

#### 调参技巧

- 点击位置尽量靠近目标中心，避免点在边缘
- 如果目标有多个不相连的部分，在每个部分各点一个前景点
- 如果结果包含了不想要的区域，在那个区域添加背景点
- `multimask_output=True` 时会生成 3 个候选，选择最符合预期的那个

### 模式二: 框提示标注

框提示是遥感场景**最推荐**的标注方式。原因: 建筑物轮廓清晰、目标边界明确、框提示精度显著高于单点。

#### 适用场景

建筑物提取、水体提取、光伏板提取等边界相对明确的目标。

#### 基本用法

坐标格式为 `[xmin, ymin, xmax, ymax]`，即左上角和右下角坐标:

```python
# 单框
masks = sam.generate_masks_by_box(box=[100, 120, 500, 480])

# 多框批量
masks = sam.generate_masks_by_boxes(boxes=[
    [100, 120, 500, 480],
    [600, 300, 900, 700],
])
```

#### 框的大小选择

框应该**略大于**目标范围:

- 太紧: SAM 可能截断目标边界
- 太松: 可能包含过多背景，导致分割不精确
- 推荐: 框的每边比目标多出 5~15 像素

#### 框 + 点联合

先框选大致范围，再用点提示修正:

```python
# 第一步: 框提示获取初始 mask
masks_init = sam.generate_masks_by_box(box=[150, 150, 450, 450])

# 第二步: 在遗漏区域添加前景点
masks_refined = sam.generate_masks_by_points(
    points=[[300, 300], [350, 350]],
    point_labels=[1, 1],
)
```

### 模式三: 文本提示标注 (Grounded-SAM)

这是本项目最有生产价值的模式。输入一段文本描述，GroundingDINO 自动检测匹配的目标，SAM 自动分割——整个过程无需人工交互。

#### 工作流

```
文本提示 → GroundingDINO 目标检测 → 生成 Bounding Box → SAM 精确分割 → Mask
```

#### 基本用法

```python
from geoai_sam import GroundedSAMWrapper

gsam = GroundedSAMWrapper(
    sam_model_type="vit_h",
    box_threshold=0.25,     # 检测框置信度阈值
    text_threshold=0.25,    # 文本匹配置信度阈值
)
gsam.set_image("西安19级.tif")

masks = gsam.segment_by_text("building")
```

#### 关键参数: 阈值调节

`box_threshold` 和 `text_threshold` 直接影响检测的召回率和精确率:

| 阈值 | 效果 | 建议场景 |
|------|------|----------|
| 0.1 ~ 0.2 | 高召回、低精确 (多检) | 不想遗漏目标，后续人工筛选 |
| 0.25 ~ 0.35 | 平衡 | **推荐默认值** |
| 0.4 ~ 0.5 | 低召回、高精确 (少检) | 精度优先，允许遗漏 |

#### 多关键词提示

用 `.` 分隔多个关键词，提高召回率:

```python
# 水体: 同时匹配 water、lake、river
masks = gsam.segment_by_text("water. lake. river")

# 建筑物: 多种表述
masks = gsam.segment_by_text("building. house. roof")
```

#### CLIP 增强

CLIP 可以对 SAM 生成的多个候选 Mask 做语义排序，选出最匹配文本的那个:

```python
masks = gsam.segment_by_text_with_clip(
    "building",
    clip_model="ViT-B-32",  # 可选 ViT-B-16, ViT-L-14
)
```

#### 遥感推荐提示词

| 目标 | 推荐提示词 |
|------|------------|
| 建筑物 | `building` 或 `building. house. roof` |
| 水体 | `water. lake. river. pond` |
| 道路 | `road. highway` |
| 植被 | `tree. forest. vegetation` |
| 光伏板 | `solar panel. solar farm` |
| 车辆 | `car. truck. vehicle` |

### 模式四: 自动分割

自动分割不需要任何提示，SAM 对全图进行网格采样，自动发现并分割所有目标。

#### 适用场景

快速了解影像中有哪些可分割目标，或作为其他模式的辅助参考。

#### 用法

```python
sam = SAMWrapper(model_type="vit_h", automatic=True)
sam.set_image("西安19级.tif")

masks = sam.generate_masks_auto(
    points_per_side=32,         # 每边采样点数
    pred_iou_thresh=0.88,       # IoU 阈值
    stability_score_thresh=0.95, # 稳定性分数
    min_mask_region_area=100,   # 最小区域面积
)
```

#### 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `points_per_side` | 32 | 每边网格点数。16=快速预览, 32=推荐, 64=精细 |
| `pred_iou_thresh` | 0.88 | 越高越严格，只保留高置信度的 mask |
| `stability_score_thresh` | 0.95 | 越高越稳定，mask 在不同扰动下越一致 |
| `min_mask_region_area` | 100 | 过滤太小的碎片 |

---

### 模式对比

| 维度 | 点提示 | 框提示 | 文本提示 | 自动分割 |
|------|--------|--------|----------|----------|
| 人工交互量 | 低 | 中 | 极低 | 无 |
| 精度 | 中 | **高** | 中 | 中 |
| 速度 | 快 | 快 | **极快** | 慢 |
| 适用目标 | 模糊目标 | **明确目标** | 批量预标注 | 探索分析 |
| 推荐度 | ★★★ | **★★★★★** | ★★★★ | ★★ |

**生产建议**: 建筑物提取首选框提示，大规模场景先用文本提示粗筛再框提示精修。

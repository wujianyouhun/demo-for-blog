## 发布建议

建议按上表顺序发布。前 3 篇建立读者认知，第 4 篇切入大影像痛点，第 5 篇讲标签落地，第 6 篇补模型选型和研究视野。

每篇可配图：

- 系统流程图：GeoTIFF -> Tile -> Prompt -> SAM -> Mask -> Vector。
- 前端截图：Web 平台左侧面板、地图叠加 Mask、整图处理进度。
- 参数表截图：模型显存需求、瓦片尺寸建议、后处理参数。
- 结果图：原图、Mask、矢量 Polygon 对比。

# 13 - 模型详解与参数说明

本文统一说明当前 GeoAI SAM 项目和 RemoteSAM 项目涉及的模型、命名、作用、权重、参数和使用方式。

## 1. 本项目模型总览

| 名称 | 英文标识 | 作用 | 使用位置 |
| --- | --- | --- | --- |
| SAM1 | `sam1` | 点/框提示分割 | `SAMWrapper`、Web 点/框标注 |
| SAM2 | `sam2` | 新版 SAM 分割能力 | `SAMWrapper` 可选 |
| SAM3 | `sam3` | 预留/兼容配置 | `SAMWrapper` 可选 |
| GroundingDINO | `GroundingDINO_SwinT/SwinB` | 文本到检测框 | `GroundedSAMWrapper` |
| CLIP | `ViT-B-32` 等 | 文本图像语义匹配 | 文本增强分割 |
| LangSAM | `samgeo.text_sam.LangSAM` | 文本提示 fallback | GroundedSAM 回退 |
| TiTiler | 非模型 | GeoTIFF 瓦片服务 | Web 影像显示 |

## 2. SAM1

SAM1 是 Meta Segment Anything Model 第一代。当前项目使用 `samgeo` 封装它，支持点提示、框提示和自动分割。

### 2.1 模型类型

| 类型 | 文件 | 预期大小 | 适用场景 |
| --- | --- | --- | --- |
| `vit_b` | `sam_vit_b_01ec64.pth` | 约 376MB | 快速验证、低显存 |
| `vit_l` | `sam_vit_l_0b3195.pth` | 约 1192MB | 默认推荐 |
| `vit_h` | `sam_vit_h_4b8939.pth` | 约 2448MB | 高精度、高显存 |

### 2.2 使用方式

```python
from geoai_sam import SAMWrapper

sam = SAMWrapper(model_type="vit_l", sam_version="sam1", automatic=False)
sam.set_image("image.tif")

masks = sam.generate_masks_by_points([[500, 500]], point_labels=[1])
masks = sam.generate_masks_by_box([100, 100, 600, 600])
```

### 2.3 关键参数

| 参数 | 说明 |
| --- | --- |
| `model_type` | `vit_b`、`vit_l`、`vit_h` |
| `sam_version` | `sam1` |
| `automatic` | 是否启用自动全图分割 |
| `checkpoint_path` | 手动指定权重文件 |
| `device` | `cuda` 或 `cpu` |
| `model_dir` | 权重目录，默认 `models/` |

### 2.4 常见错误

| 错误 | 原因 | 处理 |
| --- | --- | --- |
| 权重文件不完整 | 下载中断 | 删除 `.pth` 后重新下载 |
| CUDA out of memory | 模型过大或影像过大 | 换 `vit_b` 或使用裁剪 |
| `samgeo` 缺失 | 依赖未安装 | 安装 `samgeo` |

## 3. SAM2 / SAM3

当前项目配置中支持 `sam2` 和 `sam3` 字段。代码将 `sam2`/`sam3` 映射到 SAM2.1 相关 checkpoint：

| 类型 | 文件 |
| --- | --- |
| `vit_b` | `sam2.1_hiera_base_plus.pt` |
| `vit_l` | `sam2.1_hiera_large.pt` |
| `vit_h` | 当前复用 large 配置 |

使用方式：

```python
sam = SAMWrapper(model_type="vit_l", sam_version="sam2")
sam.set_image("image.tif")
```

SAM2 更适合进一步扩展到视频或时序场景；当前 Web 平台仍以静态 GeoTIFF 标注为主。

## 4. GroundingDINO

GroundingDINO 用于把文本提示转换为检测框，再交给 SAM 做分割。

### 4.1 作用

```text
文本提示 "building"
  -> GroundingDINO 检测建筑物框
  -> SAM 根据检测框生成 Mask
```

### 4.2 参数

| 参数 | 默认值 | 说明 |
| --- | --- | --- |
| `groundingdino_model` | `GroundingDINO_SwinT` 或配置中的 `SwinB` | 检测模型 |
| `box_threshold` | `0.25` | 检测框置信度阈值 |
| `text_threshold` | `0.25` | 文本匹配阈值 |

阈值越低召回越高但误检越多；阈值越高结果更保守。

## 5. CLIP

CLIP 用于文本和图像区域的语义匹配。在本项目中，它主要作为 Grounded-SAM 的增强能力，对候选 Mask 进行语义排序或筛选。

使用方式：

```python
masks = gsam.segment_by_text_with_clip(
    "building",
    clip_model="ViT-B-32",
)
```

## 6. LangSAM / geoai fallback

`GroundedSAMWrapper` 内部有回退策略：

1. 优先使用 `geoai.segment.GroundedSAM`。
2. 失败后尝试 `samgeo.text_sam.LangSAM`。
3. 最后回退到基础 `samgeo.SamGeo`。

这使文本标注在不同环境中有更高可用性，但不同 fallback 的结果和参数支持可能不同。

## 7. TiTiler

TiTiler 不是分割模型，而是 GeoTIFF 动态瓦片服务。

作用：

| 能力 | 说明 |
| --- | --- |
| COG/GeoTIFF 瓦片 | 浏览器按视口加载影像 |
| WebMercatorQuad | 输出 Web Mercator 瓦片 |
| 避免全图加载 | 支持大幅遥感影像浏览 |

本项目路径：

```python
from titiler.core.factory import TilerFactory
app.include_router(cog_tiler.router, prefix="/tiles/cog")
```

## 8. RemoteSAM 模型体系

RemoteSAM 是遥感统一模型，以 RES 为核心能力派生多任务。

### 8.1 RemoteSAM checkpoint

推理和评估示例使用：

```text
./pretrained_weights/checkpoint.pth
```

加载方式：

```python
model = init_demo_model("./pretrained_weights/checkpoint.pth", "cuda:0")
model = RemoteSAM(model, "cuda:0", use_EPOC=True)
```

### 8.2 Swin Transformer 初始化权重

训练需要：

```text
./pretrained_weights/swin_base_patch4_window12_384_22k.pth
```

参数：

| 参数 | 默认值 | 说明 |
| --- | --- | --- |
| `--swin_type` | `base` | Swin 规模 |
| `--window12` | false | 使用 window size 12 |
| `--pretrained_swin_weights` | Swin base 权重路径 | 初始化 backbone |

### 8.3 BERT tokenizer / 文本编码

RemoteSAM 使用 BERT tokenizer 编码文本。

相关参数：

| 参数 | 默认值 |
| --- | --- |
| `--bert_tokenizer` | `bert-base-uncased` |
| `--ck_bert` | `bert-base-uncased` |
| `max_tokens` | 20 |

源码中 `embed_sentences()` 会：

1. 使用 `BertTokenizer.encode()`。
2. 截断到 20 tokens。
3. padding 到固定长度。
4. 生成 attention mask。

## 9. LAVT / LAVTOne

RemoteSAM 的模型构建入口在 `lib/segmentation.py`。

| 名称 | 说明 |
| --- | --- |
| `lavt` | BERT 在模型外部，训练时单独构建 BERT |
| `lavt_one` | BERT/文本编码在整体模型内部，README demo 默认使用 |

默认参数：

```text
--model lavt_one
--swin_type base
--img_size 896
```

Swin base 配置：

| 配置项 | 值 |
| --- | --- |
| `embed_dim` | 128 |
| `depths` | `[2, 2, 18, 2]` |
| `num_heads` | `[4, 8, 16, 32]` |

## 10. EPOC / DirectSAM 辅助轮廓模型

RemoteSAM 的 `RemoteSAM` 类支持 `use_EPOC=True`。源码中会加载：

```text
chendelong/DirectSAM-EntitySeg-1024px-0501
```

作用：

| 用途 | 说明 |
| --- | --- |
| 轮廓辅助 | 获取实体边界或轮廓提示 |
| 检测增强 | detection/counting 中用 EPOC 结果 refine mask |
| 参数 | `EPOC_threshold=0.25` |

如果不需要 EPOC 或环境无法下载 HuggingFace 模型，可设 `use_EPOC=False`。

## 11. RemoteSAM 任务 API 参数

| 方法 | 输入 | 输出 | 参数 |
| --- | --- | --- | --- |
| `referring_seg` | image, sentence | Mask | `return_prob` |
| `semantic_seg` | image, classnames | `{class: mask}` | `return_prob` |
| `detection` | image, classnames | `{class: boxes}` | 使用 EPOC/NMS |
| `visual_grounding` | image, sentence | box | 无 |
| `multi_label_cls` | image, classnames | list | `return_prob` |
| `multi_class_cls` | image, classnames | class/None | `return_prob` |
| `captioning` | image, classnames | text | `region_split` |
| `counting` | image, classnames | `{class: count}` | 无 |

输入图像支持 PIL Image 或 numpy ndarray。若是 OpenCV 读取的 BGR 图像，需要先转 RGB。

## 12. 选型建议

| 场景 | 推荐模型 |
| --- | --- |
| Web 交互式标注 | 本项目 SAM1 `vit_l` |
| 低显存快速验证 | SAM1 `vit_b` |
| 精细边界分割 | SAM1 `vit_h` |
| 文本找目标后分割 | GroundingDINO + SAM |
| 遥感多任务研究 | RemoteSAM |
| 检测、分类、计数、描述统一处理 | RemoteSAM |
| 大幅 GeoTIFF 标签生产 | 本项目 Web + 后处理 + 矢量导出 |

## 13. 模型文件检查建议

运行前检查：

1. 模型文件是否存在。
2. 文件大小是否接近预期。
3. Python 环境是否包含 `torch`、`samgeo`、`rasterio`。
4. 文本模式是否包含 GroundingDINO / LangSAM / geoai 相关依赖。
5. RemoteSAM 是否包含 `checkpoint.pth` 和 Swin 初始化权重。

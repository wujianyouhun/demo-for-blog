## 05 - 后处理与导出

SAM 生成的原始 Mask 通常存在毛刺、小碎片、孔洞和边界抖动等问题，直接使用效果不佳。后处理是将粗糙 Mask 转化为高质量训练标签的关键步骤。

### 后处理流程

推荐的后处理顺序:

```
原始 Mask
    ↓
开运算 (去毛刺)
    ↓
去除小斑块 (面积过滤)
    ↓
孔洞填充
    ↓
闭运算 (填裂缝)
    ↓
边界平滑
    ↓
处理后 Mask
```

### 链式调用

`MaskPostProcessor` 支持链式调用，可以灵活组合各操作:

```python
from geoai_sam import MaskPostProcessor

processor = MaskPostProcessor()
clean_mask = (
    processor
    .load(raw_mask)
    .opening(radius=2)
    .remove_small_objects(min_size=200)
    .fill_holes()
    .closing(radius=3)
    .smooth(sigma=1.5)
    .get_result()
)
```

### 一键后处理

如果不确定参数，可以使用默认流程:

```python
clean_mask = MaskPostProcessor.default_pipeline(
    raw_mask,
    min_size=200,           # 最小面积阈值
    fill_holes_flag=True,   # 是否填充孔洞
    smooth_sigma=1.5,       # 平滑参数
    opening_radius=2,       # 开运算半径
    closing_radius=3,       # 闭运算半径
)
```

### 各操作参数说明

**开运算 `opening(radius)`** — 先腐蚀后膨胀，去除小的突出物和毛刺。

| radius | 效果 | 建议 |
|--------|------|------|
| 1 | 轻微 | 高分辨率影像 |
| 2 | 适中 | **推荐默认** |
| 3~5 | 较强 | 噪声较多时 |

**去小斑块 `remove_small_objects(min_size)`** — 去除面积小于阈值的连通区域。

| min_size (像素) | 效果 | 建议 |
|------------------|------|------|
| 50~100 | 保守 | 保留小目标 |
| 200 | 适中 | **推荐默认** |
| 500+ | 激进 | 只保留大目标 |

**孔洞填充 `fill_holes()`** — 填充 Mask 内部的空洞。适用于建筑物、水体等实心目标。

**闭运算 `closing(radius)`** — 先膨胀后腐蚀，填充小的间隙和裂缝。

| radius | 效果 | 建议 |
|--------|------|------|
| 1~2 | 轻微 | 边界基本完整时 |
| 3 | 适中 | **推荐默认** |
| 5+ | 较强 | 碎片化严重时 |

**平滑 `smooth(sigma, threshold)`** — 高斯滤波后二值化，平滑锯齿边界。

| sigma | 效果 | 建议 |
|-------|------|------|
| 0.5~1.0 | 轻微 | 保持边界细节 |
| 1.5 | 适中 | **推荐默认** |
| 2.0~3.0 | 较强 | 边界需要高度平滑 |

### 高级操作

**面积过滤** — 只保留指定面积范围内的区域:

```python
processor.filter_by_area(min_area=500, max_area=50000)
```

**保留最大区域** — 只保留面积最大的一个连通区域:

```python
processor.keep_largest()
```

**膨胀/腐蚀** — 手动扩大或缩小 Mask:

```python
processor.dilate(radius=2)   # 扩大
processor.erode(radius=2)    # 缩小
```

### 后处理参数调优建议

| 场景 | min_size | fill_holes | opening | closing | sigma |
|------|----------|------------|---------|---------|-------|
| 建筑物 | 200 | Yes | 2 | 3 | 1.5 |
| 水体 | 500 | Yes | 2 | 5 | 2.0 |
| 道路 | 100 | No | 1 | 2 | 1.0 |
| 植被 | 300 | No | 2 | 3 | 1.5 |
| 光伏板 | 100 | Yes | 1 | 2 | 1.0 |

### Mask 矢量化

将二值 Mask 转为 Polygon 矢量数据:

```python
from geoai_sam import MaskVectorizer

vectorizer = MaskVectorizer()
gdf = vectorizer.vectorize(
    clean_mask,
    reference_image="西安19级.tif",  # 保留地理参考
    min_area=100,                     # 过滤太小的多边形
    simplify_tolerance=0,             # 简化容差 (0=不简化)
)
```

`vectorize()` 返回一个 `GeoDataFrame`，包含 `id`、`area_pixels` 和 `geometry` 列。

### 导出格式

| 格式 | 扩展名 | 特点 | 推荐用途 |
|------|--------|------|----------|
| GeoJSON | .geojson | 纯文本，兼容性好 | Web 可视化、版本控制 |
| Shapefile | .shp | 传统 GIS 格式 | ArcGIS / QGIS |
| GeoPackage | .gpkg | 单文件，现代化 | **推荐首选** |
| GeoParquet | .parquet | 列式存储，高效 | 大数据量 |

```python
vectorizer.save("output/buildings.geojson")
vectorizer.save("output/buildings.gpkg")
vectorizer.save("output/buildings.shp")
```

### 添加自定义属性

```python
vectorizer.add_attribute("category", ["building"] * vectorizer.get_polygon_count())
vectorizer.add_attribute("confidence", [0.95, 0.88, 0.92, ...])
```

### 面积过滤

在矢量化之后进一步过滤:

```python
vectorizer.filter_by_area(min_area=500, max_area=100000)
```

### 质量评估

如果有真值标签 (Ground Truth)，可以量化评估 SAM 标签质量:

```python
from geoai_sam import QualityMetrics

# 综合评估
results = QualityMetrics.evaluate(
    prediction=pred_mask,    # SAM 生成的 Mask
    ground_truth=gt_mask,    # 真值 Mask
    include_boundary=True,   # 是否计算边界误差
)

# 打印报告
QualityMetrics.print_report(results)
```

报告包含:

| 指标 | 含义 | 优秀阈值 |
|------|------|----------|
| IoU | 交并比 | >= 0.75 |
| Dice | Dice 系数 | >= 0.85 |
| Precision | 精确率 | >= 0.80 |
| Recall | 召回率 | >= 0.80 |
| F1 | F1 分数 | >= 0.80 |
| 面积相对误差 | 面积偏差百分比 | <= 10% |
| Hausdorff 距离 | 最大边界偏差 (像素) | <= 5 |

质量等级: IoU >= 0.9 优秀, >= 0.75 良好, >= 0.5 合格, < 0.5 需改进。

批量评估多个样本:

```python
avg_results = QualityMetrics.batch_evaluate(
    predictions=[pred1, pred2, pred3],
    ground_truths=[gt1, gt2, gt3],
)
print(f"平均 IoU: {avg_results['iou']:.4f} ± {avg_results['iou_std']:.4f}")
```

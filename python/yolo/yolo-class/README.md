# YOLO v8 多模型检测演示项目

## 项目概述

本项目是一个全面的 YOLO v8 模型演示集合，展示了不同类型 YOLO v8 模型的使用方法和应用场景。项目包含多个独立的功能模块，从基础的模型信息查看到复杂的多模型检测演示。

## 项目结构

```
yolo-class/
├── README.md                           # 项目说明文档
├── README_使用说明.md                   # 详细使用说明
├── yolo_class.py                      # 模型信息查看程序
├── yolo_multi_model_demo.py           # 完整的多模型演示程序
├── quick_demo.py                      # 快速演示脚本
├── yolo_examples.py                   # 详细示例集
├── yolov8n-pose_model_info.txt        # 生成的模型信息文件
└── detection_results/                 # 检测结果输出目录
```

## 技术栈和依赖

### 核心技术
- **YOLO v8**: ultralytics 实现的 YOLO v8 框架
- **PyTorch**: 深度学习框架
- **OpenCV**: 图像处理库
- **NumPy**: 数值计算库

### 依赖库
```python
# 主要依赖
ultralytics>=8.0.0    # YOLO v8 框架
opencv-python>=4.5.0  # 图像处理
torch>=1.7.0          # 深度学习框架
numpy>=1.19.0         # 数值计算
```

### 环境要求
- Python 3.8+
- 推荐使用虚拟环境
- 可选：CUDA 支持（GPU 加速）
